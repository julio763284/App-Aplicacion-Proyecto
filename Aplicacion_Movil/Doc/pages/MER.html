Modelo Entidad Relación (MER)

👤 Actor

Administrador del sistema

🎯 Objetivo

Diseñar la estructura de la base de datos del sistema de inventario.

📝 Descripción

El modelo entidad–relación representa las entidades principales del sistema, sus atributos y las relaciones existentes entre ellas. Este modelo permite organizar la información antes de construir la base de datos.
