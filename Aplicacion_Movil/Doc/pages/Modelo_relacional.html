Modelo Relacional

👤 Actor

Administrador del sistema

🎯 Objetivo

Diseñar la estructura lógica de la base de datos del sistema de inventario.

📝 Descripción

El modelo relacional representa las tablas de la base de datos, sus atributos y las relaciones entre ellas mediante claves primarias y foráneas. Este modelo permite organizar correctamente la información del sistema.
